# philosophers
Understand threads and mutex through the famous problem of philosophers' dinner

### Prerequisites

- Sdl2
- Sdl2_ttf

## Compiling

```
make
./philo
```

![Philosopher's Dinner](https://raw.githubusercontent.com/brhamidi/philosophers/master/philosophers.png)

* Parallel computing 
* C Programming
* Shared resources
